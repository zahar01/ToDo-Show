# ToDo-show extension

This extension "todo-show" is needed to show all yours "ToDo's" in file. 

## Usage

Todo's should be described using the same template:
* `//ToDo: smth`
* `/*ToDo: smth */`

Then use *Ctrl+Shift+P* and type *"ToDo Show"*
